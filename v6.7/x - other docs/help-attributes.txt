Attributes for entries on opt-form-*.js files

        Modal (form) attributes:
                sectionPreviewKey: "sectHeader", (needs to match with the section titles on the report page)
                sectionTitle: "Report Header", (the visible title on the modal)
                defaultExcluded: true, (section loads onto form, but is hidden by default)
                defaultHidden: "sectPeff", (section is completely hidden until triggered by drop=down option with triggerSection: "sectPeff")

        Parameter Attributes:
                customText: false (specifies that an edit button should not show on the modal)
                enableSummary: true, (enables the summary checkbox)
                summaryDefault: false, (true or false, specifies whether the statament is added to summary by default)
                summaryOnChange: true, (automatically add to summary when changed from default option)
                summaryThreshold: ["xxx", "yyy"] (specify the options that DO trigger auto-add to summary)
                summaryNotThreshold: ["xxx", "yyy"] (specify the options that DO NOT trigger auto-add to summary)
                summaryAlwaysInclude: true, (to be included in summary but not show at all on modal)
                summaryExclude: ["spValves"] (specifies statements to be excluded from the summary when the summary checkbox for this parameter is activated)
                textareaSize: 3, (specify the size of the conditional textarea in the modal)
                summaryOrder: 1, (specify the summary line to be added to - will stack statements with the same number in the same row)
                options: "customtext" (enables blank text-area input field in the modal - no drop-down)
                summarytext: "..." (for use with the above to specify summary text for text-entry-only options)

        Individual option attributes:
                default: true, (within the options lines, sepcify the default option)
                summarytext: "", (within the options lines, specify different statement to be added into the summary)
